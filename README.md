# Project Name

A sleek, modern web app built with Material Design 3 (MD3).

## Features

- **Dark Mode**: Switch between light and dark themes effortlessly.
- **Responsive Design**: Looks great on desktops, tablets, and phones.
- **Material Design 3**: Clean, intuitive UI following MD3 guidelines.
- **Multilingual**: Available in multiple languages.
- **SPA**: Smooth page switching.
